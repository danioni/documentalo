# Clientes

![Lista de clientes](https://pvnhwkrgwmmrscxflqnu.supabase.co/storage/v1/object/public/docs-screenshots/screenshots/clientes-lista.png)

Los clientes son las personas que reciben servicios de tu organización. Cada cliente tiene acceso a un **portal personalizado** donde puede:

- Ver su plan de tratamiento completo
- Consultar saldo de sesiones
- Hacer reservas 24/7
- Ver historial de pagos
- Recibir confirmaciones automáticas por WhatsApp

## Modelo de datos

```typescript
interface Client {
  id: string;
  person: {
    name: string;
    lastName: string;
    email: string;
    phone?: string;
  };
  organizationId: string;
  createdAt: string;
  updatedAt: string;
  // Campos opcionales
  notes?: string;
  tags?: string[];
}
```

## Listar clientes

```bash
GET /api/v1/clients
```

### Parámetros

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `search` | `string` | Buscar por nombre o email |
| `limit` | `number` | Resultados por página (default: 20) |
| `offset` | `number` | Offset para paginación |

## Crear cliente

```bash
POST /api/v1/clients
```

### Body

```json
{
  "name": "Juan",
  "lastName": "Pérez",
  "email": "juan.perez@ejemplo.com",
  "phone": "+56912345678"
}
```

## Obtener cliente

```bash
GET /api/v1/clients/:id
```

Incluye historial de sesiones y métricas:

```json
{
  "id": "cli_789ghi",
  "person": {
    "name": "Juan",
    "lastName": "Pérez",
    "email": "juan.perez@ejemplo.com"
  },
  "stats": {
    "totalSessions": 24,
    "completedSessions": 22,
    "cancelledSessions": 2,
    "totalSpent": 480000,
    "firstSession": "2024-03-15T10:00:00Z",
    "lastSession": "2025-12-20T10:00:00Z"
  }
}
```

## Portal del cliente

Cada cliente tiene acceso a un portal personalizado mediante un token único:

```bash
GET /clientes/:token
```

El portal permite al cliente:
- Ver plan de tratamiento y sesiones pendientes
- Agendar nuevas citas en horarios disponibles
- Ver historial de sesiones completadas
- Consultar pagos pendientes y realizados

<Callout type="info">
El enlace al portal se puede enviar automáticamente por WhatsApp o email al crear el cliente.
</Callout>

## Notas importantes

<Callout type="info">
El campo `totalSpent` representa el LTV (Lifetime Value) del cliente. Este dato es clave para el lead scoring y análisis financiero de la organización.
</Callout>
