# Documéntalo

Hub central de documentación técnica en español para desarrolladores de Digitalo.

## Estructura de la documentación

| Sección | Contenido | Audiencia |
|---------|-----------|-----------|
| **Servicialo** | Estándar abierto para servicios profesionales (MCP) | Implementadores del protocolo |
| **Planifícalo** | API de gestión financiera y liquidaciones | Desarrolladores integrando finanzas |
| **Coordinalo** | API de agendamiento con 3 módulos integrados | Desarrolladores integrando la API |
| **Relaciónalo** | API de CRM y gestión de clientes | Desarrolladores integrando CRM |
| **Guías** | Tutoriales paso a paso | Desarrolladores implementando funcionalidades |
| **Integraciones** | Conexión con servicios externos | Desarrolladores configurando servicios |

## El ecosistema Coordinalo

Coordinalo es una plataforma de gestión para negocios de servicios. Se organiza en tres módulos **independientes entre sí pero que juntos cubren todo** lo que necesitas:

| Módulo | Dominio | Pregunta clave |
|--------|---------|----------------|
| **Coordinalo** | Operaciones | ¿Qué, cuándo y quién? |
| **Relaciónalo** | CRM | ¿A quién y cómo comunicamos? |
| **Planifícalo** | Finanzas | ¿Cuánto cobramos y pagamos? |
