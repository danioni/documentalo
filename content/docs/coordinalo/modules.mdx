# Módulos del Ecosistema

![Dashboard de Coordinalo](https://pvnhwkrgwmmrscxflqnu.supabase.co/storage/v1/object/public/docs-screenshots/screenshots/dashboard-admin.png)

Coordinalo se organiza en tres módulos principales que trabajan de forma integrada. Cada módulo cubre un dominio específico y mutuamente excluyente:

| Módulo | Dominio | Pregunta que responde |
|--------|---------|----------------------|
| **Coordinalo** | Operaciones | ¿Qué servicios se entregan, cuándo y por quién? |
| **Relaciónalo** | CRM | ¿A quién atendemos y cómo nos comunicamos? |
| **Planifícalo** | Finanzas | ¿Cuánto cobramos, pagamos y ganamos? |

---

## Coordinalo (Operaciones)

Gestión del día a día: sesiones, disponibilidad y catálogo de servicios.

### Alcance

| Incluye | No incluye |
|---------|------------|
| Sesiones y citas | Datos de contacto del cliente |
| Disponibilidad de profesionales | Historial de comunicaciones |
| Catálogo de servicios | Cobros y pagos |
| Reservas públicas | Campañas de marketing |
| Asignación automática | Reportes financieros |

### Funcionalidades

- **Sesiones**: Crear, confirmar, reprogramar y cancelar citas
- **Disponibilidad**: Horarios por profesional con sincronización Google Calendar
- **Catálogo**: Servicios, duraciones y precios base
- **Booking público**: Portal de reservas 24/7 sin cuenta
- **Asignación inteligente**: Sugerencias automáticas de profesional

### Rutas principales

| Ruta | Descripción |
|------|-------------|
| `/[org]/dashboard` | Dashboard administrativo |
| `/[org]/calendario` | Vista de agenda general |
| `/[org]/sesiones` | Listado de sesiones |
| `/[org]/servicios` | Catálogo de servicios |
| `/[org]/profesionales` | Gestión de profesionales |

---

## Relaciónalo (CRM)

Gestión de clientes y comunicaciones.

### Alcance

| Incluye | No incluye |
|---------|------------|
| Ficha de cliente | Agenda del profesional |
| Historial de atenciones | Disponibilidad horaria |
| Campañas y recordatorios | Facturación |
| Audiencias y segmentación | Nóminas |
| Disputas y reclamos | Estados financieros |

### Funcionalidades

- **Clientes**: Fichas completas con historial de atenciones
- **Audiencias**: Segmentación por comportamiento y atributos
- **Campañas**: WhatsApp y email masivos
- **Recordatorios**: Confirmaciones automáticas pre-sesión
- **Comunicaciones**: Log de todas las interacciones
- **Disputas**: Gestión de reclamos y resoluciones

### Métricas de cliente

| Métrica | Descripción |
|---------|-------------|
| `totalSessions` | Total de sesiones agendadas |
| `completedSessions` | Sesiones entregadas |
| `cancelledSessions` | Sesiones canceladas |
| `totalSpent` | Gasto total (LTV) |
| `attendanceRate` | Tasa de asistencia |

---

## Planifícalo (Finanzas)

Gestión financiera: ingresos, egresos y reportes.

### Alcance

| Incluye | No incluye |
|---------|------------|
| Ventas y cobros | Datos personales del cliente |
| Pagos a proveedores | Historial de comunicaciones |
| Nóminas | Gestión de citas |
| Estados financieros | Campañas de marketing |
| Comisiones | Disponibilidad horaria |

### Funcionalidades

- **Ventas**: Registro de transacciones por sesión
- **Cobros**: Integración MercadoPago, estados de pago
- **Pagos a proveedores**: Liquidaciones y transferencias
- **Nóminas**: Cálculo automático de comisiones
- **Estados financieros**: Reportes de ingresos, egresos y utilidad

### Modelo de comisiones

```typescript
interface SessionPayment {
  sessionId: string;
  totalAmount: number;           // Precio cobrado al cliente
  providerCommission: number;    // Porcentaje para el profesional
  organizationAmount: number;    // Margen de la organización
}
```

<Callout type="info">
Las comisiones se calculan automáticamente al marcar una sesión como entregada.
</Callout>

## Integración entre módulos

Los módulos comparten datos en tiempo real siguiendo el ciclo de vida del servicio:

```
┌─────────────────────────────────────────────────────────────┐
│                    CICLO DE VIDA                            │
├─────────────────────────────────────────────────────────────┤
│  RELACIÓNALO          COORDINALO           PLANIFÍCALO     │
│  (CRM)                (Operaciones)        (Finanzas)       │
│                                                             │
│  Cliente              Agenda sesión        Genera venta     │
│  solicita     →       con profesional  →   pendiente        │
│                                                             │
│  Recibe               Profesional          Procesa          │
│  confirmación ←       atiende          →   cobro            │
│                                                             │
│  Actualiza            Marca como           Calcula          │
│  historial    ←       entregada        →   comisiones       │
└─────────────────────────────────────────────────────────────┘
```

<Callout type="warning">
Los tres módulos están incluidos en todos los planes de Coordinalo. No requieren activación adicional.
</Callout>
